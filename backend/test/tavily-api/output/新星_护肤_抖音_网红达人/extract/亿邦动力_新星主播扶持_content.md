新星主播扶持计划收官 抖音电商主播上新滔滔不绝？
========================

亿邦动力 2022/06/20 16:34

“主播请就位”为直播生态良性化提供想象空间。

【亿邦原创】前有罗永浩退出社交平台，后有新东方农产品带货直播间“东方甄选”迅速走红，直播达人一浪接一浪的袭来，他们都选择了在抖音做电商。

优质内容的建设离不开创作者，在2022抖音电商生态大会上， 抖音电商总裁魏雯雯表示，“过去一年，有386万动销电商达人在平台开拓新事业。”这个数据中不仅有内容创作者、也包括主播，很多新人主播把抖音电商作为开启事业的平台之一。

而为挖掘并培养优质主播，由中青报担任支持单位、抖音电商联合巨量引擎共同打造的“主播请就位”活动6月17日完美收官。刘畊宏&张伦硕、黄圣依、沈涛、千惠、张沫凡几位明星队长的加盟让本次活动受到更多关注。

[![Image 7](https://imgs.ebrun.com/images/xqydefault.png)](https://imgs.ebrun.com/resources/2022_06/2022_06_20/2022062068416557134628751_origin.jpg)

“主播请就位”官方海报（图片来源：抖音电商）

据抖音电商统计数据，活动开启后吸引了超4000名主播报名。截止到6月17日收官，在为期一个月的活动中，新星主播开播共计1000场，主播和队长的看播人次累计超2.7亿，抖音端内#主播请就位 话题播放达到了18.7亿次。

“主播请就位”活动取得了亮眼成绩，数据背后意味着主播受到关注，而从活动流程来看，直播电商越来越职业化和专业化。而职业化的一个重要标志就是专业能力趋于标准化。尤其是在电商直播方面，能针对用户把产品故事讲得娓娓动听才是转化的关键。

随着抖音电商从“兴趣电商”升级到“全域兴趣电商”的新阶段。新人主播在更加多元丰富的场域内，通过专业训练，有机会快速晋升为小有成就的成熟、职业主播，个人与生态的价值都将进一步被放大。

**01**

**带货主播的职业“梦工厂”**

直播电商已经成为万亿产业。据iiMedia Research（艾媒咨询）数据显示，截至2021年12月份，中国直播电商相关企业数量达到5.8万家。从2019年到2021年，中国直播电商市场规模从4338亿元大涨至1.2万亿元。

电商行业与直播行业高度结合，行业市场需求大；但是主播的门槛在无形之中也逐渐增高。为了更好地培养出新一批新人主播，各大平台都纷纷采取了各种扶持举措。

一名萌新主播的“出圈”和成长要花多长的时间？每个主播的情况不同，需要的时间也自然是不同的；此外，由于各个电商平台扶持力度不一，需要的时间也存在差异。

抖音电商平台方面，推出“主播请就位”活动，通过该活动积极探索新人主播成长路径，从活动中发现优秀主播应该具备哪些职业素养和专业能力。

成为一名主播除了带货能力，更加考验的是口才、产品知识了解的能力以及自己的专业程度。

比如冉高鸣在直播间推荐着一款饮料时，用歇后语这样风趣地表达商品的优点，不仅如此，对产品的全面了解，更让冉高鸣拉近了和消费者的距离。

“我力求每一个产品，无论是吃的用的，只要是我带的，我都力求亲自尝试，我和我的同事也说，我希望我的直播间是‘磨盘张嘴，尽说实话’，我希望我的每一番话都来自我真实的表达。”冉高鸣这样说到。

对于参加此次“主播请就位”活动，冉高鸣认为，抖音电商平台给主播带来了很大的信心，平台为主播带来了三重保障，这也是他喜欢抖音电商平台带货的原因。

[![Image 8](https://imgs.ebrun.com/images/xqydefault.png)](https://imgs.ebrun.com/resources/2022_06/2022_06_20/2022062072216557135409214_origin.jpg)

主播冉高鸣抖音电商直播间截图（图片来源：抖音电商）

良好的口才和风趣幽默的语言风格，不仅能够把自己的意思清晰地传达给粉丝和消费者，还能够有效地调动现场气氛，甚至能够从正面去影响消费者。

面对队长的“考验”，主播蛋打撕葱在两分钟的产品展示中，通过古装情景剧的夸张方式惟妙惟肖的介绍卡姿兰眉笔产品，其敏捷的反应和对产品的介绍，引得直播间粉丝和队长沈涛忍俊不禁。

对于参加“主播请就位”活动的感受，蛋打撕葱表示，“这对于自己是一项挑战，面对新的内容领域，想做一个好主播其实是很难的，前期工作要做很多”，不过蛋打撕葱非常有信心的认为，剧情加带货的方式，可以让消费者更好的融入购物场景，“消费者看的开心、买的开心，是自己最大的愿望。”

明星队长沈涛对此表示，“蛋打撕葱通过对产品的详细介绍，瞬间把产品融入到情景当中去，这就是抖音电商作为兴趣电商的‘兴趣’所在。”

[![Image 9](https://imgs.ebrun.com/images/xqydefault.png)](https://imgs.ebrun.com/resources/2022_06/2022_06_20/2022062077316557135840140_origin.jpg)

（图片来源：抖音电商）

随着电商带货的火爆，主播也成为了热门职业选择。

2021年中国电商直播用户规模为4.64亿，同时根据巨量引擎数据显示，2022年直播电商行业至少有378万的人才缺口；而根据智联招聘调研数据显示，88.1%的00后正在灵活就业或愿意尝试灵活就业，其中网红/博主/达人和开店（含电商）均位居前列。

人才缺口下，是对主播专业性要求的不断提高。人社部中国就业培训技术指导中心发布的《新职业在线学习平台发展报告》显示，学历分布中，本科、专科学历共占比74%成为新职业学习主体。

为什么主播职业需要专业性和职业化？

主播的背后需要有一支具有分工协作能力的团队，包括具有选品能力，能深入理解产品的功能性能和价值亮点；同时还具有供应链管理运营能力，能全面承担起选品之后的产能协调、品控、采购和物流安排；擅长在小剧场中直播的主播还需要让直播间更有趣的故事创作者，以及布景、灯光、妆容和演播控制专家等等。

不仅如此，主播的分布结构更加趋于合理，覆盖更全面的垂直品类以及细分场景。

另外，主播是连接品牌商家与消费者的关键角色，这些都需要主播在直播带货时有正面形象和职业素养。上述这些在抖音电商“主播请就位”活动中，不仅让消费者、也让行业看到了一名优秀的主播应具备的素质。

同时，消费者也更看重“面对面”的连接。兴趣电商的繁荣，让用户对内容“穿透力”的要求越来越强，品牌与消费者的接触方式也必须更高效、互动和多元。表达力就是新的生产力，主播在直播中开始扮演越来越重要的角色。

消费者的购物习惯变了，电商主播定义了他们对美好生活的认识、对优质商品的要求。直播间重构了和品牌的关系，主播得到了此前任何商家或平台都无法企及的话语权地位。直播间成为一个全新的品牌曝光和粉丝积累路径。

这解释了为什么越来越多的主播进入了抖音电商这一新事业舞台。他们通过“内容”引发“兴趣”，重新定义消费者对美好生活的认识、对优质商品的要求。内容，商品，品牌营销在抖音电商代表的未来式想象空间这一催化剂下，主播们完成了聚变，成为了全价值的超新星。

数以百万计的主播在实现自己事业的同时，也成为了抖音兴趣电商的重要角色。

这无一不证明主播在带货中起到的关键链接作用。通过本次“主播请就位”活动，帮助众多萌新主播成长的抖音电商，成为输送主播人才的 “造梦工厂”。

[![Image 10](https://imgs.ebrun.com/images/xqydefault.png)](https://imgs.ebrun.com/resources/2022_06/2022_06_20/2022062047616557136124631_origin.png)

**02**

**萌新背后的多元支撑**

此次“主播请就位”通过重点扶持新人主播，帮助新人主播搭建起展示自我的舞台，展现了主播多样化风格，挖掘有天赋的新人主播，并提供专业的指导。

比如，抖音电商通过提供官方直播间的支持，以“麦序房”的形式将选手的直播间画面同步至抖音电商官方直播间，通过在官方直播间展示，让更多用户能够通过本次直播认识到新星主播。

[![Image 11](https://imgs.ebrun.com/images/xqydefault.png)](https://imgs.ebrun.com/resources/2022_06/2022_06_20/2022062009016557136932393_origin.jpg)

抖音电商“麦序房”直播间（图片来源：抖音电商）

据了解，“麦序房”是一个能够串联多个抖音直播间的产品能力，可以借助一个主直播账号，串联不同直播间的内容，按照预设的节目单依次在同一个账号下展现。实际上，“麦序房”功能是实现了抖音直播的“转播”功能，将麦序主播的直播内容实时“转播”到主直播间。

不仅如此，除了在活动中增强玩法的多样性、强化内容及流量对主播的作用力，抖音电商更在节目之外给与支持，包括百万流量扶持，抖音端流量入口等优质资源，着力培养有能力、有潜力的电商主播。

除了在上述资源层面的支持，抖音电商还搭建了新老主播交流的舞台，除了在直播中呈现专业的指导，也邀请了刘芳等电商作者为新人主播讲理论课等，全方位提升主播专业能力。

在全民皆可播的时代，抖音电商通过“主播请就位”活动，吸引优质人才入场，通过优化B、C端的服务体验，打造优质主播渠道和内容曝光。

据统计，5月20日，刘畊宏通过直播连麦各战队队长，邀请大家在#主播请就位 话题挑战下分享身边的好物，吸引各类主播积极分享，截止发稿前，话题已有18.7亿次播放。

[![Image 12](https://imgs.ebrun.com/images/xqydefault.png)](https://imgs.ebrun.com/resources/2022_06/2022_06_20/2022062090316557137406284_origin.png)

（图片来源：抖音电商）

亿邦动力了解到，“主播请就位”新人主播扶持活动收官期与618好物节时间节点前后衔接，新人主播可以天然承接618大促带来的流量。也就是说，抖音电商寄希望于在平台重点资源扶持和借助于618本身天然自带的流量和热度双重助力之下，能够在今年的618好物节活动中，培养出一批新星主播。

这些举措，对于抖音电商主播来说不仅仅是能力、职业素养的提升，同时还是自己的展示。同时，借助“主播请就位”，抖音电商挖掘新人主播天赋，提供专业的指导，为电商行业输送更多出众的主播人才，最终服务消费者。

**03**

**兴趣致广泛，推陈出新滔滔不绝**

越来越多新面孔出现在抖音电商平台各式各样的直播间中，这表明，众多萌新主播已经将抖音电商作为开启事业的首选平台之一。抖音电商不断探索，也是带货主播吐故纳新，包罗万象的过程。

以千惠战队的萌新主播wuli米言为例，他在抖音电商平台从事护肤品种草。“我像是一个不断往前冲往前跑、去挑战去探索的一个主播，之所以选择抖音平台参加主播请就位活动，是想让更多人认识我，也非常感谢关注我的粉丝们，有你们的支持才让我一路走到现在的。这对我来说是一个全新的挑战，希望自己可以成为一个技多不压身的斜杠青年。”wuli米言表示。

对于未来规划和选择抖音电商平台，wuli米言表示，“自己是一个脾气暴躁、但内心非常温暖、口嫌体正直的大哥哥形象，（希望自己）可以成为一个让人很信任的主播，希望大家信任我，我可以为大家去把关，把更多美好的、有意思的东西，给大家做分享。”

对于萌新主播三娘小喵酱而言，她将所有时间都献给了直播平台。

“把直播作为我的事业，并且坚持到底，选择了就不会放弃，有责任心和恒心。”她表示，之所以选择在抖音电商平台做直播，是因为“抖音平台是非常公正的平台，只要我们每一个创作者够努力，在（抖音电商）优质的土壤当中，我们都可以发光发热。”

“参加主播请就位对于新人主播是非常好的展示机会。”对于未来，“希望能在抖音电商平台上建立自己的品牌，把美和优质生活品质内容分享给粉丝们。”三娘小喵酱说道。

对此，“梦想大使”刘畊宏也回忆起自己直播的初衷。在互联网直播竞争激烈的丛林法则中脱颖而出，刘畊宏创造了一名长期身处逆境者的翻身传奇。他表示，“综艺节目有时间限制，但在直播间，我可以认真的把健身的方法传递出去。”

一直以来，抖音电商平台致力于支持不同阶段、类型的电商主播发展，打造多元的主播生态。依托愈发丰富的直播生态和资源扶持，也预示着电商主播职业也迎来发展的好时机。

比起授人以鱼，主播请就位活动更在于授人以渔。抖音电商不仅会在流量上给予萌新主播扶持，在重要大促活动节点，提供选品、内容等方面的建议。

正如沈涛说的那样，“抖音电商‘主播请就位’的活动，正是为了帮助更多人圆梦”，而这也正响应了本次活动的宗旨。

不管内容生态还是商业生态，想要发展壮大，走得长远，需要有正向、激励人心的精神内核。一个良性发展的商业生态，既能带来丰厚的经济支撑，又能达到实现人生价值的目标。无疑，抖音电商平台提供了双重机会。
