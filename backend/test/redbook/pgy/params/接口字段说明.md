### 🔥 核心漏斗数据

| 字段名 | 中文含义 | 数据示例 | 业务意义 |
|--------|----------|----------|----------|
| `impNum` | 曝光数 | 32166 | 内容被展示次数 |
| `readNum` | 阅读数 | 2194 | 点击进入阅读次数 |
| `likeNum` | 点赞数 | 178 | 用户点赞次数 |
| `favNum` | 收藏数 | 21 | 用户收藏次数 |
| `cmtNum` | 评论数 | 9 | 用户评论次数 |
| `shareNum` | 分享数 | 4 | 用户分享次数 |
| `followCnt` | 新增关注 | 11 | 通过此笔记新增粉丝 |

### 漏斗转化率计算

```
曝光 → 阅读: readNum / impNum = 2194 / 32166 = 6.82%
阅读 → 点赞: likeNum / readNum = 178 / 2194 = 8.11%
阅读 → 收藏: favNum / readNum = 21 / 2194 = 0.96%
阅读 → 评论: cmtNum / readNum = 9 / 2194 = 0.41%
阅读 → 分享: shareNum / readNum = 4 / 2194 = 0.18%
阅读 → 关注: followCnt / readNum = 11 / 2194 = 0.50%
```
