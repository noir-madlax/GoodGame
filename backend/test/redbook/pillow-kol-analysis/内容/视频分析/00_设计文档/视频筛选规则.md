# 视频筛选规则

## 一、筛选目标

从31位入选KOL中（排除纯图文博主），每人选取 **5个典型视频** 用于内容能力分析。

---

## 二、筛选策略

### 2.1 核心规则

- **不区分广告与非广告**：统一按互动量排序
- 按 **互动总量** 降序排列，选取每位KOL的TOP5视频
- 纯图文博主从本次视频分析中排除

```
互动总量 = 点赞数 + 收藏数 + 评论数
```

### 2.2 筛选优先级

| 优先级 | 筛选条件 | 数量 | 目的 |
|--------|----------|------|------|
| 1 | 互动最多 TOP5 | 5篇 | 分析达人最佳内容表现 |
| 补充 | 视频不足时全部选取 | ≤5篇 | 确保每位KOL都有分析样本 |

### 2.3 特别说明

- 广告视频同样纳入分析，作为商业内容创作能力的参考
- 分析时会记录视频是否为广告（`is_advertise`字段）
- 不对广告/非广告内容做评分权重区分

---

## 三、边界情况处理

### 3.1 视频数量不足

对于视频少于5个的KOL，全部选取其所有视频。

| KOL名称 | 视频数 | 处理方案 |
|---------|--------|----------|
| 格丽乔 | 2个 | 全部选取 |
| 汤圆不要加冰 | 2个 | 全部选取 |
| Qw1ko | 4个 | 全部选取 |

### 3.2 纯图文博主

**汤痴婆** 是纯图文创作者，无视频内容，从本次视频分析中排除。

## 四、筛选SQL

```sql
WITH selected_kols AS (
    SELECT kol_id, kol_name, fans_count_current
    FROM gg_pgy_kol_analysis_result
    WHERE post_frequency_pass = true 
      AND comment_gt_20_pass = true 
      AND read_fans_ratio_pass = true
),
kols_with_videos AS (
    SELECT sk.kol_id, sk.kol_name, sk.fans_count_current, COUNT(n.note_id) as video_count
    FROM selected_kols sk
    JOIN gg_pgy_kol_notes n ON sk.kol_id = n.kol_id
    WHERE n.is_video = true
    GROUP BY sk.kol_id, sk.kol_name, sk.fans_count_current
    HAVING COUNT(n.note_id) > 0
),
ranked_videos AS (
    SELECT 
        kv.kol_id,
        kv.kol_name,
        n.note_id,
        n.title,
        n.is_advertise,
        n.read_num,
        n.like_num,
        n.collect_num,
        n.comment_num,
        n.publish_date,
        (n.like_num + n.collect_num + COALESCE(n.comment_num, 0)) as total_interact,
        ROW_NUMBER() OVER (
            PARTITION BY kv.kol_id 
            ORDER BY (n.like_num + n.collect_num + COALESCE(n.comment_num, 0)) DESC
        ) as rank
    FROM kols_with_videos kv
    JOIN gg_pgy_kol_notes n ON kv.kol_id = n.kol_id
    WHERE n.is_video = true
)
SELECT * FROM ranked_videos WHERE rank <= 5 ORDER BY kol_name, rank;
```
