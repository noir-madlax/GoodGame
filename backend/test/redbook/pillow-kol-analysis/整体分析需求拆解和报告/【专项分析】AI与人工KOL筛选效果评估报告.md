# AI与人工KOL筛选效果评估报告

> **报告类型**：AI效果分析专项报告  
> **分析对象**：全量257位候选KOL  
> **评估维度**：准确率、召回率、F1分数、一致性分析  
> **生成日期**：2025年12月11日

---

## 一、评估框架设计

### 1.1 评估目标

本报告旨在评估AI数据筛选系统与人工评估之间的**一致性**和**互补性**，为后续KOL筛选流程优化提供数据支撑。

### 1.2 评估定义

| 概念 | 定义 |
| ---- | ---- |
| **AI选中** | 三项硬指标全部通过：阅读/粉丝>30% + 评论>20条 + 每周发帖>3篇 |
| **人工认可** | XQX或WYY至少一人给出正面反馈（ok/✔/还可以/看价格等） |
| **人工否定** | XQX和WYY均给出负面反馈（nok/不ok/数据差等），或仅一人反馈且为否定 |

### 1.3 人工反馈语义分类规则

| 分类 | 关键词/模式 | 示例 |
| ---- | ----------- | ---- |
| **明确认可** | ok、OK、✔、可以、还可以、不错 | "ok，看价格"、"✔"、"还可以" |
| **条件认可** | 看价格、可以考虑、备选 | "1000以内可以考虑"、"可以备选" |
| **明确否定** | nok、不ok、不OK、数据差、数据不好 | "nok"、"数据太差了 不OK" |
| **保留意见** | 一般、数据一般、同质化 | "一般，中医一般带不了价格较高的产品" |
| **风险提示** | 已合作竞品、广告痕迹重、人群不匹配 | "已经种草过竞品了" |

## 三、准确率与召回率分析

### 3.1 核心指标计算

以**人工认可**作为"正确答案"（Ground Truth），评估AI筛选的效果：

| 指标 | 公式 | 计算 | 结果 |
| ---- | ---- | ---- | ---- |
| **真阳性 (TP)** | AI选中 ∩ 人工认可 | 18位 | - |
| **假阳性 (FP)** | AI选中 ∩ 人工否定 | 6位 | - |
| **假阴性 (FN)** | AI淘汰 ∩ 人工认可 | 112位 | - |
| **真阴性 (TN)** | AI淘汰 ∩ 人工否定 | 42位 | - |

### 3.2 评估指标

| 指标 | 公式 | 结果 | 解读 |
| ---- | ---- | ---- | ---- |
| **准确率 (Precision)** | TP / (TP + FP) | **75.0%** | AI选中的KOL中，75%被人工认可 |
| **召回率 (Recall)** | TP / (TP + FN) | **13.8%** | 人工认可的KOL中，仅13.8%被AI选中 |
| **F1分数** | 2×P×R / (P+R) | **23.4%** | 准确率与召回率的调和平均 |
| **特异度 (Specificity)** | TN / (TN + FP) | **87.5%** | AI正确排除否定KOL的能力 |

### 3.3 指标解读

```
准确率高 (75%) + 召回率低 (13.8%) = AI筛选策略偏保守

┌─────────────────────────────────────────────────────────┐
│                    人工认可的130位KOL                    │
│  ┌─────────────┐                                        │
│  │ AI选中18位  │  AI漏选112位（召回率仅13.8%）          │
│  │  (TP=18)    │  (FN=112)                              │
│  └─────────────┘                                        │
└─────────────────────────────────────────────────────────┘

AI选中的24位KOL（有人工反馈）：
┌─────────────────┬───────────────┐
│ 人工认可 18位   │ 人工否定 6位  │
│  (75%准确)      │  (25%误判)    │
└─────────────────┴───────────────┘
```

**核心发现**：
- AI筛选**准确率较高**，选出的KOL大部分被人工认可
- AI筛选**召回率很低**，大量人工认可的KOL被AI淘汰
- 这意味着AI的硬指标筛选**过于严格**，遗漏了许多有价值的KOL

## 五、差异原因深度分析

### 5.1 AI筛选的系统性偏差

| 偏差类型 | 影响 | 占遗漏比例 | 说明 |
