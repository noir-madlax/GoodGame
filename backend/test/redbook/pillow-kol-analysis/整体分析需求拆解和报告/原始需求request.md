在这些博主中，有一些已经合作了的博主，列表在执行表中。
是否可以先从这些博主的表现数据中进行一些维度分析。

第一方面：通用数据维度：

1. 粉丝数量的增长趋势
2. 发帖频率
3. 阅读、点赞、评论的平均值&中位数（过去一个月）
   这些是从数据层面上可以筛选的维度

第二方面：
我们自己人工在筛选的时候，会关注数据+内容层面：
数据层面：

1. 粉丝数据vs阅读、评论、互动数据，在非投流情况下，单篇阅读数据>粉丝数据的30%；单篇评论数据>20条
2. 数据波动性，月内不止一篇爆文（避免单篇爆文拉高平均数）
3. 数据趋势，粉丝数据趋势、阅读评论互动的数据趋势
4. 发文频率，最好每周>3篇以上

内容层面：

1. 博主美不美，家居风格是不是高级（内容是软装而不是硬装，家居风格比较温馨而不是工业风等）
2. 剪辑情况，是不是有多种不同剪辑风格
3. 口播能力，能否说清楚卖点
