```

┌───────────────────┐                 ┌───────────────────┐
│  MCN候选名单      │                 │  入选KOL名单      │
│    257 位KOL      │                 │    32 位KOL       │
└────────┬──────────┘                 └────────┬──────────┘
         │                                      │
         ▼                                      ▼
┌───────────────────┐                 ┌───────────────────┐
│  API数据采集      │                 │  视频内容获取     │
│  (蒲公英+小红书)  │                 │  (每人TOP5视频)   │
└────────┬──────────┘                 └────────┬──────────┘
         │                                      │
         ▼                                      ▼
┌───────────────────┐                 ┌───────────────────┐
│  六维度数据分析   │                 │  AI多模态分析     │
│  粉丝/发帖/互动等 │                 │  (Gemini 2.5)     │
└────────┬──────────┘                 └────────┬──────────┘
         │                                      │
         ▼                                      ▼
┌───────────────────┐                 ┌───────────────────┐
│  硬指标过滤       │                 │  三维度评分       │
│  3项必须全部达标  │                 │  形象/剪辑/口播   │
└────────┬──────────┘                 └────────┬──────────┘
         │                                      │
         ▼                                      ▼
    32位KOL入选                           最终排名输出
```

### 1.2 六大数据维度分析

| 维度                    | 分析内容           | 判断标准                 | 符合要求KOL数 |
| ----------------------- | ------------------ | ------------------------ | ------------- |
| **需求1：粉丝增长趋势** | 30天内粉丝变化情况 | 增长>5%为上升期          | 104位 (43.3%) |

### 1.3 硬指标筛选条件

为确保入选KOL具备基本的数据表现能力，我们设定了三项**必须同时满足**的硬指标：

| 条件编号       | 筛选条件               | 业务含义                   |
| -------------- | ---------------------- | -------------------------- |
| **条件1-1a**   | 单篇阅读 > 粉丝数×30%  | 内容触达能力，确保非刷量   |
| **条件1-1b**   | 单篇评论 > 20条        | 粉丝互动活跃度，有真实讨论 |
| **条件1-2**    | 每周发帖 > 3篇         | 内容更新稳定性，保持活跃   |

### 1.4 内容层面评估标准

#### 1.4.1 分析工具

| 项目         | 说明                                      |

| 节奏把控 | 整体节奏、节奏变化、BGM配合    |

| 表达能力 | 逻辑结构、语言流畅、用词准确     |

#### 1.4.3 评分标准

| 分数    | 等级 | 说明                         |

#### 1.4.4 推荐等级划分

| 综合评分      | 推荐等级        | 合作建议                 |

## 二、数据筛选过程

### 2.1 筛选漏斗

从MCN机构提供的**257位候选KOL**出发，经过硬指标层层筛选，最终**32位KOL**进入内容分析阶段。

```
全量候选KOL: 257位 ← MCN提供名单
        │
        ▼
条件1-1a：阅读>粉丝30%
        │ 淘汰 91位
        ▼
      166位
        │
        ▼
条件1-1b：评论>20条
        │ 淘汰 27位
        ▼
      139位
        │
        ▼
条件1-2：每周>3篇
        │ 淘汰 107位
        ▼
      32位 → 最终入选 (占比 12.5%)
```

### 2.2 被淘汰KOL的主要原因分析

| 淘汰阶段            | 淘汰数量 | 主要原因                           |
| ------------------- | -------- | ---------------------------------- |
| 阅读/粉丝比例不达标 | 91位     | 内容触达能力不足，可能存在刷量嫌疑 |
| 评论数不达标        | 27位     | 粉丝互动活跃度低，缺乏真实讨论     |
| 发帖频率不达标      | 107位    | 更新频率过低，活跃度不足           |

**关键发现**：发帖频率是最严格的筛选条件，仅18.5%的KOL能够满足"每周≥3篇"的要求，这反映了小红书平台上高频稳定输出内容的KOL相对稀缺。
