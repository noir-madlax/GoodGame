# 附录：数据口径说明

| 指标 | 数据来源 | 说明 |
|------|----------|------|
| 粉丝增长率 | kol_fans_trend API | 30天内粉丝变化率 |
| 发帖频率 | kol_note_list API | 30天内笔记数/4.3周 |
| 近7天活跃 | kol_data_summary_v1 | activeDayInLast7 字段 |
| 阅读/评论均值 | kol_note_list + 原生接口 | 30天内非广告笔记 |
| 超越同行比例 | kol_note_rate API | readMedianBeyondRate 等 |
| 阅读>粉丝30% | 自行计算 | 非广告笔记阅读/粉丝>0.3 |
| 评论>20条 | 原生接口 comments_count | 30天内有评论>20条的笔记 |
| 爆文定义 | 自行计算 | 互动 > 互动中位数 × 3 |
| 互动趋势 | kol_core_data API | 30天engage数据线性回归 |
