* **潜力人选**：基于"高流量+高互动"标准，筛选出 **50位** 具备强带货潜力的优质达人，其中"成分护肤郭昱君"等头部达人兼具流量与极高赞评比。

## 3. 核心量化指标说明

### 基础互动指标 (Available ✅)

* **`fans_count` (粉丝数)**: KOL的粉丝总量，代表账号的基础影响力。
* **`play_count / vv` (播放量)**: 视频的观看次数，衡量内容的曝光广度。
* **`digg_count` (点赞数)**: 用户对内容的基本认可。
* **`comment_count` (评论数)**: 用户的深度参与意愿。
* **`share_count` (分享数)**: 内容的口碑传播能力。
* **`collect_count` (收藏数)**: 内容的实用价值和收藏意愿。

### 漏斗转化指标 (Funnel Metrics)

从达人粉丝到最终互动的完整转化链路：

1. **粉丝 → 播放转化率** = `播放量 / 粉丝数`
   * 衡量粉丝的内容触达率和观看意愿
   * 健康范围：5%-30% (粉丝粘性强)

2. **播放 → 点赞转化率** = `点赞数 / 播放量`
   * 衡量内容质量对观众的吸引力
   * 健康范围：1%-5% (内容受认可)

3. **点赞 → 深度互动率** = `(评论+分享+收藏) / 点赞数`
   * 衡量粉丝从浅层认可到深度参与的意愿
   * 健康范围：>8% 为高互动

### 衍生效率指标 (Calculated)

* **`like_comment_ratio` (赞评比)**: 点赞÷评论。>50 为高质量内容，衡量粉丝对内容的认同度。
* **`interaction_rate` (互动率)**: (评论+分享+收藏)÷点赞。>0.08 表示强互动，衡量粉丝参与度。
* **`share_ratio` (分享占比)**: 分享÷点赞。>0.02 为易传播，衡量口碑传播意愿。
* **`collect_ratio` (收藏占比)**: 收藏÷点赞。>0.05 (5%) 为高价值，衡量内容的实用价值（如教程类）。

### 热度评估指标

* **`heat_index` (热度指数)**: (总互动÷播放量)×1000。>10 为爆款。
* **`virality_score` (传播力指数)**: 分享数÷点赞数×100。>5% 为强传播。
* **`play_like_ratio` (播赞比)**: 播放÷点赞。10-30 为优秀（完播率替代指标）。
