## 核心性价比指标

### 1. expect_cpe（预期每互动成本）

**业务价值**: ⭐⭐⭐⭐⭐ **极高** - ROI核心预估指标

**CPE = Cost Per Engagement (每次互动成本)**

| 字段 | 说明 | 计算公式 | 业务意义 |
|------|------|---------|---------|
| `cpe_1_20` | 1-20秒视频的预期CPE | 视频报价 / 预期互动数 | ✅ **核心** - 短视频互动成本 |
| `cpe_21_60` | 21-60秒视频的预期CPE | 视频报价 / 预期互动数 | ✅ **核心** - 中视频互动成本 |
| `cpe_60` | 60秒以上视频的预期CPE | 视频报价 / 预期互动数 | ✅ **核心** - 长视频互动成本 |

**数值单位**: 元（已转换，不是分）  
**互动定义**: 点赞 + 评论 + 分享

#### 业务应用

**CPE越低 = 性价比越高**

| CPE范围 | 评级 | 说明 |
|---------|------|------|
| <5元 | ⭐⭐⭐⭐⭐ 极优 | 顶级性价比 |
| 5-10元 | ⭐⭐⭐⭐ 优秀 | 性价比很好 |
| 10-20元 | ⭐⭐⭐ 良好 | 行业平均水平 |
| 20-50元 | ⭐⭐ 一般 | 略高于平均 |
| >50元 | ⭐ 较差 | 需谨慎评估 |

**示例解读**:
```json
"cpe_1_20": "1187"  // 1187元每次互动
```
如果报价1500元，预期互动数 ≈ 1500 / 11.87 ≈ 126次

---

### 2. expect_cpm（预期千次曝光成本）

**业务价值**: ⭐⭐⭐⭐⭐ **极高** - 品牌曝光效率指标

**CPM = Cost Per Mille (千次展示成本)**

| 字段 | 说明 | 计算公式 | 业务意义 |
|------|------|---------|---------|
| `cpm_1_20` | 1-20秒视频的预期CPM | (视频报价 / 预期播放量) × 1000 | ✅ **核心** - 短视频曝光成本 |
| `cpm_21_60` | 21-60秒视频的预期CPM | (视频报价 / 预期播放量) × 1000 | ✅ **核心** - 中视频曝光成本 |
| `cpm_60` | 60秒以上视频的预期CPM | (视频报价 / 预期播放量) × 1000 | ✅ **核心** - 长视频曝光成本 |

**数值单位**: 元（已转换）

#### 业务应用

**CPM越低 = 品牌曝光越划算**

| CPM范围 | 评级 | 说明 | 应用场景 |
|---------|------|------|---------|
| <10元 | ⭐⭐⭐⭐⭐ 极优 | 顶级曝光效率 | 新品推广、品牌认知 |
| 10-30元 | ⭐⭐⭐⭐ 优秀 | 优质曝光渠道 | 品牌建设 |
| 30-50元 | ⭐⭐⭐ 良好 | 行业平均水平 | 常规投放 |
| 50-100元 | ⭐⭐ 一般 | 曝光成本较高 | 精准人群 |
| >100元 | ⭐ 较贵 | 需结合转化考量 | 高净值人群 |

**对比参考**:
- 信息流广告CPM：30-80元
- 开屏广告CPM：100-200元
- KOL内容营销CPM：10-50元（更具信任感）

**示例解读**:
```json
"cpm_1_20": "21123"  // 211.23元/千次曝光
```
如果报价1500元，预期播放量 ≈ (1500 / 211.23) × 1000 ≈ 7,100次

---

### 3. expect_vv（预期播放量）

**业务价值**: ⭐⭐⭐⭐⭐ **极高** - 传播力核心指标

| 字段 | 说明 | 业务意义 |
|------|------|---------|
| `value` | 预期播放量（VV = Video Views） | ✅ **核心** - KOL内容传播能力 |

**数值单位**: 次

#### 业务应用

| 播放量级 | KOL类型 | 说明 |
|---------|---------|------|
| <10万 | 初级达人 | 小规模传播 |
| 10-50万 | 腰部达人 | 中等影响力 |
| 50-100万 | 准头部达人 | 较强传播力 |
| 100-500万 | 头部达人 | 强传播力 |
| >500万 | 超头达人 | 超强传播力 |

**与粉丝数关系**:
- 健康比例：播放量 ≈ 粉丝数的5-10%
- 优质内容：播放量 ≈ 粉丝数的10-30%
- 爆款内容：播放量 > 粉丝数的50%

**示例解读**:
```json
"expect_vv": { "value": 710100 }  // 预期71万播放
```
若粉丝1571万，播放率 ≈ 4.5%（符合健康比例）

### 4. hot_item（热门作品数量）

**业务价值**: ⭐⭐⭐⭐ **高** - 爆款制作能力

| 字段 | 说明 | 业务意义 |
|------|------|---------|
| `value` | 近期热门作品数量 | ✅ **重要** - 爆款内容产出能力 |

**数值单位**: 个

#### 业务应用

| 热门作品数 | 评级 | 说明 |
|-----------|------|------|
| 0-2 | ⭐ 一般 | 稳定性较低 |
| 3-5 | ⭐⭐ 良好 | 有爆款能力 |
| 6-10 | ⭐⭐⭐ 优秀 | 较强爆款能力 |
| 11-20 | ⭐⭐⭐⭐ 很强 | 稳定爆款产出 |
| >20 | ⭐⭐⭐⭐⭐ 极强 | 顶级内容能力 |

**热门标准**: 通常指播放量超过平均值2-3倍的作品

**示例解读**:
```json
"hot_item": { "value": 1200 }  // 近期有1200个热门作品（异常高）
```
*注：此数值可能是累计值或其他统计口径*

## 性价比综合评估模型

### 模型1：互动性价比评分

```
互动性价比得分 = (行业平均CPE / 该KOL的CPE) × 100
```

**示例**:
- 行业平均CPE: 15元
- 该KOL的CPE: 11.87元  
- 互动性价比得分 = (15 / 11.87) × 100 = 126分

**评分标准**:
- >150分：超高性价比 ⭐⭐⭐⭐⭐
- 120-150分：高性价比 ⭐⭐⭐⭐
- 80-120分：性价比一般 ⭐⭐⭐
- <80分：性价比较低 ⭐⭐

---

### 模型2：品牌曝光性价比评分

```
曝光性价比得分 = (行业平均CPM / 该KOL的CPM) × 100
```

**示例**:
- 行业平均CPM: 40元
- 该KOL的CPM: 211.23元
- 曝光性价比得分 = (40 / 211.23) × 100 = 18.9分

**说明**: 曝光成本过高，品牌建设阶段不推荐

---

### 模型3：综合ROI预估

```
预期ROI = (预期播放量 × 转化率 × 客单价) / 合作成本
```

**示例计算**:
- 预期播放量: 710,100
- 行业平均转化率: 0.5%
- 客单价: 200元
- 合作成本: 3,200元

```
预期销售额 = 710,100 × 0.5% × 200 = 710,100元
预期ROI = 710,100 / 3,200 = 221.9倍
```

**ROI评级**:
- >10倍：超高回报 ⭐⭐⭐⭐⭐
- 5-10倍：高回报 ⭐⭐⭐⭐
- 3-5倍：良好回报 ⭐⭐⭐
- 1-3倍：保本微利 ⭐⭐
- <1倍：亏损 ⭐

## 业务决策应用

### 场景1：品牌曝光型投放
**目标**: 最大化品牌曝光
- **核心指标**: CPM、expect_vv
- **决策**: 选择CPM<30元且播放量>50万的KOL

### 场景2：互动种草型投放
**目标**: 引发用户互动和讨论
- **核心指标**: CPE、hot_item
- **决策**: 选择CPE<15元且热门作品>5个的KOL

### 场景3：效果转化型投放
**目标**: 直接带货转化
- **核心指标**: ROI预估、CPE
- **决策**: 综合考量CPE和转化能力，预期ROI>3

### 场景4：测试性投放
**目标**: 小成本测试KOL效果
- **决策**: 选择1-20s短视频（最低价格），观察实际CPE和播放量

---

## 数据使用注意事项

⚠️ **预期数据 ≠ 保证数据**
- 所有指标均为"预期"（expect），实际效果受多因素影响
- 内容质量、发布时间、平台推荐都会影响最终表现

✅ **建议做法**:
1. 首次合作：预留20-30%的效果浮动空间
2. 长期合作：基于历史数据优化预期
3. 对比评估：横向对比多个KOL的性价比指标
4. 动态调整：根据实际效果调整合作策略
