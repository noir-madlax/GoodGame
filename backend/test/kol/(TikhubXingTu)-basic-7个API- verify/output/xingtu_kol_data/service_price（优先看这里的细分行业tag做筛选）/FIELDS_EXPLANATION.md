## price_info 数组详解（核心商务数据）

每个price_info对象代表一种服务类型的报价信息：

### 报价核心字段

| 字段 | 类型 | 说明 | 业务意义 |
|------|------|------|---------|
| `price` | Integer | 当前报价（单位：分） | ✅ **核心** - 实际合作价格 |
| `origin_price` | Integer | 原价（单位：分） | ✅ 未打折前的价格 |
| `settlement_type` | Integer | 结算类型 | ✅ **重要** - 定价模式 |
| `settlement_desc` | String | 结算描述 | ✅ 定价模式说明 |
| `has_discount` | Boolean | 是否有折扣 | ✅ 是否参与平台优惠 |

**价格换算**: `价格（元） = price / 100`

#### 结算类型 (settlement_type)

| 代码 | 说明 | 业务意义 |
|------|------|---------|
| `1` | CPM（千次曝光成本） | ⚠️ 按曝光量计费，风险较高 |
| `2` | 固定价格 | ✅ **最常见** - 一口价模式 |
| `3` | CPC（点击成本） | ⚠️ 按点击计费 |
| `4` | CPA（行动成本） | ⚠️ 按转化计费 |

## 业务价值分析

### 🔥 核心商务决策指标

1. **报价梯度分析**
   - 1-20s视频：基础价格
   - 21-60s视频：中档价格（通常是基础的2倍）
   - 60s+视频：高端价格（通常是基础的2-3倍）

2. **性价比计算**
   - 单位时长成本 = 价格 / 视频时长
   - 20秒视频：75元/秒
   - 60秒视频：53元/秒（更划算）

3. **服务多样性**
   - 服务类型越多，合作灵活性越高
   - 是否支持直播、图文等多种形式

### ⚡ 成本预估

基于price字段可快速计算：
- **小规模测试**: 1-2个短视频，预算2000-5000元
- **中等规模**: 3-5个视频（含长短视频），预算1-2万元
- **大规模投放**: 10+视频或短视频+直播，预算5万+

### 📊 行业对标

根据粉丝量级和报价区间：
- **腰部KOL** (100万-500万粉丝): 1000-5000元/条
- **头部KOL** (500万-1000万粉丝): 5000-2万元/条
- **超头KOL** (1000万+粉丝): 2万-10万+元/条

### ⚠️ 注意事项

1. **价格只是起点**: 实际合作价格需商务洽谈
2. **季节波动**: 电商大促期价格可能上浮20-50%
3. **内容深度**: 深度测评可能额外收费
4. **修改次数**: 多次修改可能产生额外费用
5. **独家合作**: 独家期可能溢价30-100%
