## 🔍 多账号匹配分析

### 为什么会有多个匹配账号？

#### 1️⃣ **同名账号现象**（最常见）
不同的人使用相似或相同的昵称。例如：
- 搜索"程十安an"返回27个账号
- 包括"程十安an"(2213万粉丝)、"程十安来撸妆"(2.3万粉丝)、其他同名账号等

**比例**: 100%的达人都遇到多账号情况

#### 2️⃣ **同一达人的多个账号**
达人可能运营多个账号：
- 主账号（粉丝最多）
- 分类账号（护肤号、彩妆号、带货号）
- 合作品牌账号

**示例**: 
- "贾乃亮" vs "贾乃亮国货美妆"
- "朱梓骁" vs "朱梓骁-HBN"

#### 3️⃣ **模仿/山寨账号**
热门达人会被模仿，账号名相似但粉丝差距巨大

#### 4️⃣ **昵称变化**
达人可能改名或添加后缀：
- "骆王宇" → 完全匹配
- "美妆春蚕" → "美妆春蚕（双11囤货版）"
- "豆子" → "唐豆子"

## 🎯 筛选策略说明

### 智能评分系统

我们开发了一个**综合评分算法**来自动筛选真正的达人主账号：

```
总分 = 昵称匹配度 × 40% + 粉丝数 × 30% + 点赞数 × 20% + 作品数 × 10%
```

#### 各维度说明：

| 维度 | 权重 | 说明 | 归一化方法 |
|------|------|------|------------|
| **昵称匹配度** | 40% | 最关键指标，完全匹配优先 | 字符串相似度算法 |
| **粉丝数** | 30% | 影响力最直观体现 | 相对最大值归一化 |
| **点赞数** | 20% | 内容受欢迎程度 | 相对最大值归一化 |
| **作品数** | 10% | 活跃度指标 | 相对最大值归一化 |

### 为什么这样设计？

1. **昵称优先**（40%）
   - 名字匹配是身份识别的核心
   - 完全匹配的优先级最高

2. **粉丝数次之**（30%）
   - 主账号通常粉丝最多
   - 有效区分主号和小号

3. **点赞数考量**（20%）
   - 反映内容质量和影响力
   - 排除僵尸账号

4. **作品数辅助**（10%）
   - 活跃度指标
   - 权重较低避免刷量账号干扰

### 匹配可信度分级

| 评分范围 | 可信度 | 建议 |
|----------|--------|------|
| > 70% | **高** | 可直接使用 ✅ |
| 50%-70% | **中等** | 建议人工复核 ⚠️ |
| < 50% | **低** | 需要人工验证 ❌ |
