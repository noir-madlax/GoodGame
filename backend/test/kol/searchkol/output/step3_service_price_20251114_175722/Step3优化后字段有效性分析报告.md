### 2. 价格数据覆盖率

#### 问题发现
**66.7% (4/6) 的KOL返回的price_info为空**

可能原因：
1. ❓ KOL未在星图平台设置报价
2. ❓ KOL暂时关闭了接单（is_open=false）
3. ❓ 某些KOL类型不通过星图接单（如自营品牌）
4. ❓ 需要特定权限或Cookie才能查看报价

#### 有价格数据的KOL特征
- **美妆成分专家莫同学** (950,547粉)
  - 提供完整视频服务报价
  - 价格梯度合理：200元-300元
  - 额外提供广告投放服务

- **蝶莲娜-专注护肤** (932,416粉)
  - 仅提供广告投放服务
  - 价格极低：0.1元（可能是测试价格）

### 2. 可用筛选维度

#### ✅ 高可用性筛选
```sql

WHERE 'beauty-护肤' = ANY(industry_tags);

-- 按是否有报价筛选

WHERE price_info_count > 0;
```

#### ⚠️ 低可用性筛选
```sql
-- 按具体价格筛选（数据稀疏）
SELECT * FROM gg_xingtu_kol_price
WHERE video_1_20s_price IS NOT NULL
AND video_1_20s_price BETWEEN 10000 AND 50000;
```

### 3. 数据完整性建议

#### 立即可用 ✅
- **行业匹配**: 使用`industry_tags`筛选目标行业KOL

#### 需要补充 📋

- **服务范围**: 无法仅通过API确定所有KOL的服务类型
