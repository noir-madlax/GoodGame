## 核心转化能力指标

### 1. 销售转化指标

#### avg_sales_amount_range（平均销售额区间）

**业务价值**: ⭐⭐⭐⭐⭐ **极高** - 带货能力核心指标

| 字段类型 | 说明 | 业务意义 |
|---------|------|---------|
| String | 90天内平均每场/每条带货视频的销售额区间 | ✅ **核心** - 直接衡量变现能力 |

**区间示例**:
| 销售额区间 | 评级 | KOL类型 | 适合品类 |
|-----------|------|---------|---------|
| <1万 | ⭐ 初级 | 新手带货达人 | 低客单价产品 |
| 1万-5万 | ⭐⭐ 入门 | 成长期达人 | 大众消费品 |
| 5万-10万 | ⭐⭐⭐ 合格 | 腰部带货达人 | 中端产品 |
| 10万-50万 | ⭐⭐⭐⭐ 优秀 | 头部带货达人 | 品牌产品 |
| 50万-100万 | ⭐⭐⭐⭐⭐ 卓越 | 顶级带货达人 | 高端/大众爆品 |
| >100万 | 🏆 超级 | 超头带货达人 | 全品类 |

**示例解读**:
```json
"avg_sales_amount_range": "50万-100万"
```
- 近90天平均每次带货销售额在50-100万之间
- **佣金预估** (假设10%佣金率): 5万-10万元/次
- **ROI预估**: 若合作费3万元，佣金净利2-7万元

| <1万 | ⭐ | 引流能力较弱 |
| 1万-5万 | ⭐⭐ | 引流能力一般 |

#### component_click_rate_range（组件点击率区间）

**业务价值**: ⭐⭐⭐⭐⭐ **极高** - 内容转化效率指标

**点击率 = 点击量 / 播放量**

| 字段类型 | 说明 | 业务意义 |
|---------|------|---------|
| String | 组件点击率百分比区间 | ✅ **核心** - 内容带货转化率 |

**区间评级**:
| 点击率区间 | 评级 | 说明 | 应用建议 |
|-----------|------|------|---------|
| <2% | ⭐ | 转化能力弱 | 仅适合品牌曝光 |
| 2%-4% | ⭐⭐ | 转化能力一般 | 大众化产品 |
| 4%-6% | ⭐⭐⭐ | 转化能力良好 | 行业平均水平 |
| 6%-8% | ⭐⭐⭐⭐ | 转化能力优秀 | 精准带货达人 |
| 8%-10% | ⭐⭐⭐⭐⭐ | 转化能力极强 | 顶级带货达人 |
| >10% | 🏆 | 超强转化 | 爆款制造机 |

**行业对比**:
- 普通内容点击率: 0.5-1%
- 广告内容点击率: 1-2%
- 优质带货内容: 4-8%
- 顶级带货内容: 8-15%

**示例解读**:
```json
"component_click_rate_range": "6%-8%"
```

- 远高于行业平均，内容种草能力强

### 3. 商业价值指标

#### gpm_range（GPM区间）

**业务价值**: ⭐⭐⭐⭐⭐ **极高** - 单位播放价值核心指标

**GPM = Gross Profit per Mille (千次播放毛利)**

| 字段类型 | 说明 | 业务意义 |
|---------|------|---------|
| String | 每千次播放带来的毛利（元） | ✅ **核心** - 内容商业化效率 |

**区间评级**:
| GPM区间 | 评级 | 说明 | 合作建议 |
|---------|------|------|---------|
| <10 | ⭐ | 商业价值低 | 不建议带货合作 |
| 10-30 | ⭐⭐ | 商业价值一般 | 低价商品 |
| 30-50 | ⭐⭐⭐ | 商业价值良好 | 大众商品 |
| 50-100 | ⭐⭐⭐⭐ | 商业价值优秀 | 品牌商品 |
| 100-150 | ⭐⭐⭐⭐⭐ | 商业价值极高 | 高端商品 |
| >150 | 🏆 | 超高商业价值 | 全品类 |

**GPM计算示例**:
```
播放量: 71万
销售额: 50万
产品毛利率: 30%
毛利: 15万

GPM = (15万 / 71万) × 1000 = 211.3元
```

**示例解读**:
```json
"gpm_range": "150以上"
```
- 每1000次播放带来150+元毛利
- 若播放71万，预期毛利 ≈ 10.65万元
- 扣除合作费3万元，净利润 ≈ 7.65万元

## 综合转化能力评估模型

### 模型1：带货能力评分

```
带货能力得分 = (
  销售额权重 × 40% +
  点击率权重 × 30% +
  GPM权重 × 20% +
  播放量权重 × 10%
) × 100
```

**权重评分标准**:
- 销售额50-100万: 1.0
- 点击率6-8%: 1.0
- GPM 150+: 1.0
- 播放量150万: 1.0

**评分示例**:
```
= (1.0 × 40% + 1.0 × 30% + 1.0 × 20% + 1.0 × 10%) × 100
= 100分（满分）
```

---

### 模型2：ROI预估公式

```
预期ROI = 销售额 × 毛利率 / (合作费 + CPC × 点击量)
```

**示例计算**:
- 预期销售额: 75万元（取50-100万中值）
- 产品毛利率: 30%
- 合作费: 3万元
- CPC: 9元
- 点击量: 7.5万（取5-10万中值）

```
成本 = 3万 + 9 × 7.5万 = 70.5万元
毛利 = 75万 × 30% = 22.5万元
ROI = 22.5 / 70.5 = 0.32
```

**结论**: 此场景下ROI<1，需重新评估合作模式或产品定价

---

### 模型3：带货达人等级判定

基于5个核心指标：

| 指标 | 初级 | 中级 | 高级 | 顶级 |
|------|------|------|------|------|
| 销售额 | <5万 | 5-50万 | 50-100万 | >100万 |
| 点击率 | <4% | 4-6% | 6-8% | >8% |
| GPM | <30 | 30-100 | 100-150 | >150 |
| CPC | >10 | 5-10 | 3-5 | <3 |
| 播放中位数 | <50万 | 50-100万 | 100-200万 | >200万 |

**综合判定**: 满足3个以上指标即可定级

## 业务决策指南

### 何时选择此KOL？

✅ **强烈推荐** (满足以下任3条):
- 销售额 >50万
- 点击率 >6%
- GPM >100
- 推荐价格区间匹配产品定价
- 目标品类有成功案例

⚠️ **谨慎评估** (满足以下任2条):
- 销售额 <10万
- 点击率 <4%
- CPC >10元
- 品类不匹配
